🎮 Mario Game (JavaScript)

📌 Sobre o projeto

Este projeto é um jogo simples inspirado no clássico Mario, desenvolvido com HTML, CSS e JavaScript puro (Vanilla JS).

O objetivo principal deste projeto não é recriar o jogo original com fidelidade, mas sim servir como ambiente de aprendizado e prática, permitindo explorar e testar diferentes funcionalidades do JavaScript na prática.

---

🎯 Objetivo

Este projeto foi criado para:

- Praticar lógica de programação
- Trabalhar com manipulação do DOM
- Entender eventos do teclado
- Simular mecânicas básicas de jogos (como pulo e colisão)
- Experimentar e evoluir conhecimentos em JavaScript

---

⚙️ Funcionalidades

- 🕹️ Controle do personagem
- ⬆️ Sistema de pulo
- 🚧 Obstáculos no cenário
- 💥 Detecção de colisão
- 🛑 Game Over ao colidir

---

🧠 Como funciona

O funcionamento do jogo é baseado em algumas lógicas principais:

🔹 Movimento

O personagem responde a eventos do teclado, permitindo ações como pular.

🔹 Gravidade (simulada)

O pulo do personagem é controlado por alterações na posição vertical, simulando uma física básica.

🔹 Colisão

O jogo verifica constantemente a posição do personagem em relação aos obstáculos.
Se houver interseção entre eles, o jogo é encerrado.

🔹 Loop do jogo

Um loop contínuo (usando "setInterval" ou "requestAnimationFrame") atualiza:

- posição dos elementos
- estado do jogo
- verificações de colisão
